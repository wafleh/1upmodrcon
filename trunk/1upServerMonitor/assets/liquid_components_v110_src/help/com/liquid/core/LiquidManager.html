loadSharedSkin()method
public static function loadSharedSkin(id:String, url:String, context:LoaderContext = null):void

Load an external bitmap skin file and shared it for the Liquid Components. Note : By default, LiquidManager attach the default BitmapData class in the library : Liquid_focusRect

Parameters
id:String — String indentifier, this property take the folowing value :

idDescription
focusRectFocus rectangle of all components
 
url:String — URL of the bitmap file
 
context:LoaderContext (default = null) — A LoaderContext object, which has properties that define the following :
  • Whether or not Flash Player should check for the existence of a policy file upon loading the object
  • The ApplicationDomain for the loaded object
  • The SecurityDomain for the loaded object

For complete details, see the description of the properties in the LoaderContext class.

Example 
LiquidManager.loadSharedSkin("focusRect","./skin/mySkin.png");
